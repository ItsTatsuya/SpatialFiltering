# Repo for doing Spatial Filtering Experiments

## Image Quality Metrics

| PSNR Value(db) | Image Quality |
|----------------|---------------|
| 20-25          | Poor          |
| 25-30          | Fair          |
| 30-35          | Good          |
| 35-40          | Very Good     |
| 40-45          | Excellent     |
| 45-50          | Outstanding   |
